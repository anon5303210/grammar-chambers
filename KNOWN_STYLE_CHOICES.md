# Known style choices (house style, not universal grammar)

Where authorities or writers legitimately differ, the app either skips the issue or adopts
a house rule and labels it. Every adoption is listed here. Authority hierarchy: Judge
Grant's consistent practice (chambers style only) → Garner's Redbook → Garner's Modern
English Usage (GMEU) → Chicago Manual of Style (CMOS 17/18).

| Issue | House choice | Basis | Drilled? |
|---|---|---|---|
| Serial comma | Required | Grant (uniform), Redbook, CMOS | Yes — binary |
| Restrictive *which* | Use *that* | Grant (uniform), Redbook; GMEU calls which acceptable BrE | Yes — binary, labeled convention |
| 's after singular s-nouns (Congress's) | Required | Grant, SCOTUS modern practice, CMOS 7.17 | Yes — binary |
| Plural-form entity names (Oakes Farms') | Bare apostrophe | Grant, CMOS | Yes — binary |
| Date-as-adjective ("a March 20, 2012, email") | Both commas | Grant (recurring), CMOS 6.38 | Yes — binary, labeled convention |
| Capital after colon (full sentence follows) | Capitalize | SCOTUS uniform; CMOS allows either | Yes — labeled convention |
| Numbers | Spell ≤ one hundred; "percent" as word | Grant; CMOS humanities style | Yes — labeled convention |
| Em dash spacing | Unspaced (closed) | Grant, SCOTUS, CMOS | Yes — binary |
| Closed compounds: decisionmaking, factfinder, caselaw, lookback | Closed in own prose | Grant (uniform in her prose) | Light — labeled chambers style |
| Covid-19 (own prose) | Initial cap only | Grant | Not drilled (informational) |
| "(quotation omitted)" family, never "(cleaned up)" | Grant's formula | Grant (uniform) | Not drilled (citation-adjacent, out of scope) |
| Comma after short introductory adverb | NOT drilled as binary | Grant varies | Taught as tendency only |
| Contractions | NOT errors; negatives never contracted in Grant's prose | Writer-split | Taught in myths module |
| Generic he/she/singular they | NOT drilled | Writer-split, evolving usage | Excluded |
| Subjunctive was/were | Counterfactual *were* preferred; NOT drilled as binary where corpus varies | GMEU; Grant varies | One item, accepts context |
| Notional "none are" | Both accepted | Grant uses both; GMEU: none = both OK | Distractor-guard only |
| % vs. percent | House: "percent" | Grant; Kagan uses % | Labeled convention |
| Split infinitives, terminal prepositions, initial And/But, passive voice | Explicitly taught as MYTHS | GMEU, Garner; corpus practice | Yes — myths module |
| Deliberate fragments | Licensed style; drills train recognizing intent | Grant/SCOTUS practice | Yes — in proofreading (never planted as "errors") |
